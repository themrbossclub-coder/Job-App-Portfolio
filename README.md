# JobApp 🚀

A modern job listing and recruitment mobile application built with **Flutter**. JobStall connects job seekers with employers, making it easy to discover opportunities and manage applications on the go.

---

## 📱 Features

### 👤 For Job Seekers

* Browse available jobs by category, location, or keywords
* Apply to jobs directly from the app
* Save jobs for later
* Track application status
* Create and manage user profile

### 🏢 For Employers

* Post new job listings
* Manage job applications
* View candidate profiles
* Edit or delete job posts

### 🔐 Authentication

* User registration and login
* Secure authentication system
* Role-based access (Job Seeker / Employer)

## ⚙️ Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/jobstall.git
```

2. Navigate to the project folder:

```bash
cd jobstall
```

3. Install dependencies:

```bash
flutter pub get
```

4. Run the app:

```bash
flutter run
```

---

## 🔧 Configuration

### Firebase Setup

1. Create a Firebase project
2. Add Android/iOS apps
3. Download and place:

   * `google-services.json` (Android)
   * `GoogleService-Info.plist` (iOS)
4. Enable Authentication & Firestore
